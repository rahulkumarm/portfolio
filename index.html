Solving Travelling Salesman Problem using an Enhanced Ant Colony Optimization Algorithm

IJCSE July 2019

A modified pheromone rule is used as a key parameter in solving the TSP. The experimental medium takes a graph of cities and their distances, followed by solving it using the proposed approach. The data analytics of maximum, mean and minimum costs of the graph are analyzed using R tool. The experimental results prove that the proposed enhanced ant colony algorithm effectively solves Travelling Salesman Problem.
